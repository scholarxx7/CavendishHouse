# Cavendish Hostel | Patna Science College

![Cavendish Hostel Banner](https://images.unsplash.com/photo-1562774053-701939374585?ixlib=rb-1.2.1&auto=format&fit=crop&w=1920&q=80)

## Overview

This is the repository for the official website of **Cavendish House**, a heritage hostel of **Patna Science College**, Patna University. Established in 1928 and inaugurated by Lord Irwin, the hostel has been a home to some of Bihar's most brilliant scientific minds for nearly a century.

The website serves as a digital portal for prospective students, current residents, and alumni, providing information about the hostel's history, facilities, rules, and admission process.

## Features

-   **Responsive Design:** Fully responsive layout built with Tailwind CSS, ensuring a seamless experience across mobile and desktop devices.
-   **Historical Timeline:** A dedicated Legacy page detailing the rich history of the hostel from 1927 to the present day.
-   **Alumni Showcase:** A section honoring distinguished alumni like Dr. Vashishtha Narayan Singh, Prof. H.C. Verma, and others.
-   **Information Portal:** Comprehensive details on admission criteria, fee structure, and hostel rules.
-   **Interactive Elements:** Smooth scrolling, mobile navigation menu, and dynamic UI components using Lucide icons.

## Technology Stack

-   **HTML5:** Semantic markup for structure.
-   **Tailwind CSS:** Utility-first CSS framework for styling (via CDN).
-   **Vanilla JavaScript:** Lightweight interaction logic.
-   **Lucide Icons:** Modern, consistent iconography.
-   **Google Fonts:** Typography using 'Inter' and 'Merriweather'.

## Key Pages

1.  **Home (`index.html`):** Overview, Hero section, Facilities, Fees, and Contact info.
2.  **Legacy (`legacy.html`):** In-depth history and interactive timeline of the hostel.
3.  **Alumni (`alumni.html`):** Profiles of notable past residents and college alumni.
4.  **Rules (`rules.html`):** detailed code of conduct and anti-ragging policies.
5.  **Privacy Policy (`privacy.html`):** Data collection and usage policies.

## Setup & Deployment

This is a **static website**, meaning it requires no backend server setup.

### Running Locally
1.  Clone the repository:
    ```bash
    git clone https://github.com/scholarxx7/CavendishHouse.git
    ```
2.  Open `index.html` in any modern web browser.

### Deployment
This site is ready to be deployed on any static hosting service:
-   **GitHub Pages:** Go to Settings > Pages > Select `main` branch.
-   **Netlify / Vercel:** Simply connect the repository and deploy.
